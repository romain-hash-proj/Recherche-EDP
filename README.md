# Recherche-EDP
Projet relatif au parcours recherche, mon sujet porte sur l'étude des EDP hyperboliques et leurs résolutions.
Vous trouverez ici le code python et le rapport préliminaire (Janvier 2026).

rech_LF_erreur est le programme qui permet d'afficher l'évolution de l'erreur en fonction du nombre de points choisi pour la résolution numérique.
rech_LF_pression_zr est le programme qui précède l'étude de l'erreur, il s'agit du squelette initial, permettant de visualiser la pression (ainsi que densité et vitesse); il affiche également la vitesse du choc et les valeurs exactes de Z_r.
